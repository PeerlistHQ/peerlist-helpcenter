### Sign up for Peerlist

You can <a target="_blank" href={PEERLIST_EXTERNAL_LINKS.signup}>sign up for Peerlist</a> with your Google account <a target="_blank" href={GOOGLE_SSO}>Google SSO </a> or Email+Password

**Username**: Select your desired username aka profile handle. Usernames are subjects to availability. Please refer to our Terms for username.

<Frame>
<img src='/images/create-profile/welcome-modal.png'  />
</Frame>
**LinkedIn Profile URL**: You can share your LinkedIn profile URL to autofill some details like: first/last name, profile picture, work experience, education, certifications, projects, etc. This options makes the onboarding super fast and is recommended.

Please note that:

* We can autofill only if your LinkedIn profile is publicly available.

* You can use Autofill with LinkedIn only 3 times.

* This is a one-time import. That means it does not sync your LinkedIn profile and Peerlist profile and won‚Äôt automatically update the information in future.

* We try our best to get the most updated information from your Peerlist profile. But do cross-check the imported data is correct. If not; make the appropriate correction manually.

**Enter manually**: In case we couldn‚Äôt import your data from LinkedIn, please proceed with the Enter Manually option.

## Profile FAQs

  <Accordion title="Why my profile got suspended?">

    * Using individual profile as company profile with brand name and logo.

    * Multiple instances of spamming. This might include violating our rules and guidelines for Scroll, Spotlight, Jobs, etc.
